# Implementação

>[!NOTE] 
 Relatar o processo de implementação do problemas, incluindo as
 ferramentas e bibliotecas utilizadas
>

O projeto será desenvolvido na linguagem **C++**, utilizando os paradigmas da Programação Orientada a Objetos.

Para a criação da interface gráfica, janelas, renderização dos objetos e captura de eventos (teclado, mouse), será utilizado o framework **Qt 6**. A escolha se dá pela sua robustez, ampla documentação e integração nativa com C++, além de possuir um sistema de sinais e slots que facilita a comunicação entre objetos.

O ambiente de desenvolvimento integrado (IDE) utilizado será o **Qt Creator**, que já vem com todas as ferramentas necessárias para compilar e depurar o projeto. O controle de versão será feito com **Git** e o repositório hospedado no **GitHub**.
